# **DVWA Lab**

Repository ghi lại quá trình học và thực hành **Web Application Penetration Testing** trên môi trường **DVWA (Damn Vulnerable Web Application)**.

Mục tiêu của dự án không chỉ là khai thác các lỗ hổng mà còn tìm hiểu nguyên nhân, phân tích mã nguồn và nghiên cứu các phương pháp khắc phục để hiểu rõ cách xây dựng ứng dụng web an toàn.

---

## Mục tiêu

* Tìm hiểu các lỗ hổng phổ biến trong ứng dụng web.
* Thực hành kiểm thử xâm nhập web theo từng chủ đề.
* Phân tích nguyên nhân gây ra lỗ hổng.
* Tìm hiểu cách khắc phục và phòng chống.
* Ghi chép toàn bộ quá trình học dưới dạng tài liệu.

---

## Môi trường Lab

| Thành phần    | Mô tả         |
| ------------- | ------------- |
| Máy chủ       | Ubuntu Server |
| Máy tấn công  | Kali Linux    |
| Web Server    | Apache        |
| Ngôn ngữ      | PHP           |
| Cơ sở dữ liệu | MariaDB       |
| Ứng dụng      | DVWA          |

---

## Tài liệu

* `architecture.md` — Kiến trúc của môi trường Lab.
* `web-tour.md` — Khám phá toàn bộ giao diện và chức năng của DVWA.
* `setup.md` — Quá trình cài đặt và cấu hình.
* `vulnerabilities/` — Ghi chép chi tiết từng lỗ hổng.

## Trong mỗi folder **vulnerabilities**
Trả lời được 6 câu hỏi:
- Đây là lỗ hổng gì?
- Đây là lỗ hổng gì?
- Điều kiện để khai thác?
- Mình đã khai thác như thế nào?
- Kết quả đạt được?
- Cách phòng chống?
